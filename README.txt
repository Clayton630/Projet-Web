Avant toute chose, assurez vous d'avoir Python installé sur votre machine. Pour lancer le site en local, veuillez vous rendre dans un premier temps sur la branche main et télécharger tout son contenu, avant de le compresser. Une fois fait, installez uWamp (ou équivalent) puis lancez phpmyadmin depuis là (laissez l'identifiant et mot de passe par défaut). Sur phpmyadmin, créez une nouvelle base, allez sur cette base et faites importer, puis sélectionnez le fichier "retoursdb.sql" depuis le dossier décompressé précédemment. Ensuite, ouvrez le dossier en question, puis ouvrez un terminal à la racine de ce dossier, créez un vent et activez-le. Installez toutes les librairies du requirements.txt avec cette commande : "pip install -r requirements.txt".

C'est bon, vous êtes prêt ! Désormais, chaque fois que vous voudrez lancer votre site, il faudra lancer uwamp, activer votre venv, puis entrer la commande python run.py. (il est possible d'automatiser le tout via un script si nécessaire).
